Playwright Job Scraper
======================

This project demonstrates a simple Playwright scraper that writes job listings to a CSV file.

Prerequisites:
1. Node.js installed (64-bit).
2. Internet connection.

Setup & Run:
------------

1. Open a terminal and navigate to this folder.
2. Install dependencies:
   npm install
3. Run the scraper:
   npm start

Output:
-------
- playwright_jobs.csv: Contains job Title, Company, and Location.
